# SkinChest (스킨상자)
[![.NET](https://img.shields.io/badge/.NET-8.0-512BD4?logo=dotnet)](https://dotnet.microsoft.com/)
[![Windows](https://img.shields.io/badge/Platform-Windows-0078D6?logo=windows)](https://www.microsoft.com/windows)
[![License](https://img.shields.io/badge/License-GPL3.0-green.svg)](LICENSE)

> 마인크래프트 스킨을 보다 쉽게 변경하세요!

## 개요
C# WinForm을 통해 스킨을 로컬에 저장하고 로그인한 계정으로 선택한 스킨을 적용하는 프로그램입니다.  
[CmlLib.Core.Auth.Microsoft](https://www.nuget.org/packages/CmlLib.Core.Auth.Microsoft/)를 이용해 로그인하고,  
[MojangAPI](https://www.nuget.org/packages/MojangAPI/)를 통해 로그인한 계정을 통해 스킨을 적용합니다.

## 기능
1. 여러게의 스킨을 한번에 관리할 수 있습니다
2. 추가한 스킨을 더블클릭하면 3D 형식으로 미리보기가 가능합니다.
3. `%appdata%SkinChest`으로 추가한 스킨들이 로컬 파일로 추가됩니다.
